Cookie.txt の取得方法

1

拡張機能をインストール

Chrome ウェブストアから「Get cookies.txt LOCALLY」をインストールします。

Chrome ウェブストアを開く
2

シークレットモードで有効化

拡張機能の設定で「シークレットモードでの実行を許可する」を有効にします。

  1. Chrome の拡張機能アイコンをクリック
  2. 「Get cookies.txt LOCALLY」の「︙」をクリック
  3. 「拡張機能を管理」を選択
  4. 「シークレット モードでの実行を許可する」をON
3

シークレットモードでYouTubeにログイン

シークレットモード（Ctrl+Shift+N / Cmd+Shift+N）で新しいウィンドウを開き、YouTubeにログインします。

⚠️ メンバーシップに加入しているアカウントでログインしてください
4

Cookie.txtをエクスポート

YouTubeのページで拡張機能アイコンをクリックし、「Export」ボタンでCookie.txtをダウンロードします。

  1. YouTube のページを開いた状態で
  2. 拡張機能アイコン → 「Get cookies.txt LOCALLY」
  3. 「Export」または「Download」をクリック
  4. ダウンロードした Cookie.txt をこのページでアップロード
💡 なぜシークレットモード？

通常モードではCookieが自動的にローテーション（更新）されるため、エクスポートしたCookieがすぐに無効になることがあります。シークレットモードを使用することで、この問題を回避できます。
